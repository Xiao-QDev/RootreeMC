# Network网络处理层已完成
## ServerAddr.go - 地址管理
- IP 和端口管理 - 完整的地址结构体
- 地址解析 - 从字符串解析地址
- 地址验证 - 有效性检查
- 监听功能 - 支持 TCP 监听
- 连接功能 - 支持 TCP 连接
- 链式调用 - 便捷的地址配置
- 默认配置 - Minecraft 标准端口 25565

## network.go - 网络通信
- 创建网络连接 - 创建 TCP 连接
- 数据包发送 - Send/SendPacket
- 数据包接收 - Read/ReadPacket
- Varint 编解码 - Minecraft 协议支持
- 并发安全 - 使用 Mutex 保护
- 错误处理 - 完善的错误处理机制
